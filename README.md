## Features

- **Create Charts** - Design beautiful bar and pie charts with an intuitive editor
- **Customize Colors** - Pick custom colors for each data segment
- **Animated Charts** - Export charts as animated GIFs
- **Gallery** - Save and browse your chart collection
- **Download** - Export charts as PNG or GIF images

## Tech Stack

- **React 19** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **React Router** - Client-side routing
- **Lucide React** - Icons
- **gif.js** - GIF generation
- **Vitest** - Testing

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Run tests
npm test

# Build for production
npm run build
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm test` | Run tests in watch mode |
| `npm run test:run` | Run tests once |
| `npm run test:coverage` | Run tests with coverage |
| `npm run deploy` | Deploy to GitHub Pages |
